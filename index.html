Introduction to Data Analytics
A foundational course covering the basics of data analysis, including data collection, cleaning, and visualization.
SQL for Data Analysis
Teaches SQL (Structured Query Language), which is essential for managing and querying databases.
Excel for Data Analysis
Focuses on using Microsoft Excel for data manipulation, analysis, and visualization, including advanced functions.
Python for Data Analysis
Covers the use of Python programming language, particularly libraries like Pandas and NumPy, for data manipulation and analysis.
R for Data Science
Teaches the R programming language for statistical analysis, data visualization, and data manipulation.
